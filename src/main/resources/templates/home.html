Endpoints

GET /api/ping
Returns a simple JSON pong message.
GET /api/debug/headers
Request/response header dump (JSON). One secret (DEBUG_HEADERS_ACCESS_TOKEN): put it in header X-Debug-Token, or if you omit the header, in query ?debug-token=… (same value either way). Query is handy in the browser but leaks more easily — prefer the header. Without a valid value, you get 403.
